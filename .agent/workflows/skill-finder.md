# Skill Finder

Use this workflow to discover and install new AI skills.

## 1. Context
Activate this workflow when the user asks "How do I do X" or "Is there a skill for X".

## 2. Skill Source
Read: `C:\Users\ICe\.gemini\antigravity\skills\.agents\skills\find-skills\SKILL.md`

## 3. Action
- Formulate keywords based on the user's requirements (e.g., `react testing`).
- Suggest the user run the `npx skills find [keyword]` command to search.
- Explain the function of the discovered skills.
